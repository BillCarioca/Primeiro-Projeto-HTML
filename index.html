Estrutura Básica do HTML

(volta)

Na estrutura básica1 do documento HTML começamos com a tag HTML e terminamos com a marca /HTML. Nela contém um cabeçalho que descreve o título da página que será exibido na aba do navegador, delimitado pelas tags HEAD e /HEAD. Também dentro das tags HTML e /HTML, um corpo no qual se encontra o conteúdo da página, que será exibido no navegador do cliente, delimitado pelas tags BODY e /BODY.

Tags de HTML

(volta)

Usamos tags para informar ao navegador a estrutura do site HTML, assim quando se escrevemos um código em HTML, as tags usadas serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página no navegador do cliente. A principal característica das tags é estarem sempre dentro dos sinais de chevron (sinal de “maior que” e “menor que”), ou seja: < >.

As tags HTML são divididas em dois tipos: as que precisamos fechar e as que não precisamos fechar. As primeiras possuem a sintaxe TAG /TAG, já as últimas possuem como estrutura TAG/. Além disso, uma mesma tag pode receber um ou mais atributos, que possuirá um valor que modifica sua estrutura ou funcionalidade.

Atributos das Tags

(volta)

Utilizamos os atributos para personalizar as tags, modificando sua estrutura ou funcionalidade. Igualmente, os atributos são utilizados para atribuir uma classe ou id a um elemento por exemplo. A maioria das tags tem seus próprios atributos. Contudo, existem alguns atributos genéricos que podem ser utilizados na maioria das tags HTML.

Exemplos:

  • class =”…“ – Atribui uma classe ao elemento (uma classe pode ser utilizada para um ou mais elementos);
  • id =”…“ – Atribui um id ao elemento (um id deve ser único, ou seja atribuído a um único elemento);
  • lang =”…” – Define o idioma principal do elemento;
  • title =”…” – Define o título do elemento;
  • width =”…” – Define uma largura para o elemento;
